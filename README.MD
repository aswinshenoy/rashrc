
# Rashrc - Customized Bash for Higher Productivity

## Features
1. Shortcut Commands
2. Colorful Bash
3. Timestamp for each line

## How to Install?
1. Clone the repository into the home folder.
2. Run ``$ ./install_rashrc.sh``

## Shortcut Commands
| Regular Bash | Rashrc |
|--|--|
| sudo apt-get *[* install *]* *[* remove *]* *[* upgrade *]* | *[* install *]* *[* remove *]* *[* upgrade *]* |
| cd *[* .. *]* *[* ../.. *]* *[* ../../.. *]*  | *[* .. *]* *[* ... *]* *[* .... *]* |
